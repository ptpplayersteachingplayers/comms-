Enterprise communication platform with SMS, Voice, Microsoft Teams, HubSpot integration, and intelligent automations.

== Description ==

PTP Comms Hub Enterprise is a comprehensive communication platform designed for PTP Soccer Camps. It provides:

* **SMS & Voice Messaging** - Send text and voice messages via Twilio
* **Contact Management** - Organize and manage customer contacts
* **Automations** - Trigger messages based on events (orders, reminders, follow-ups)
* **Templates** - Pre-built professional message templates
* **HubSpot Integration** - Sync contacts and log activities
* **Microsoft Teams Integration** - Get real-time notifications
* **Inbox** - Manage two-way conversations
* **Campaigns** - Send bulk messages to segmented audiences
* **WooCommerce Integration** - Auto-sync orders and registrations

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ptp-comms-hub/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to PTP Comms > Settings to configure Twilio, HubSpot, and Microsoft Teams
4. Start creating templates and automations

== Frequently Asked Questions ==

= What integrations are required? =

Twilio is required for SMS and voice. HubSpot and Microsoft Teams are optional but recommended.

= How do automations work? =

Automations trigger messages based on specific events like order completion or upcoming events. You can set delays and choose which template to send.

= Can customers reply to messages? =

Yes! The plugin supports two-way messaging. Replies appear in the Inbox.
